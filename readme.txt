Этот проект отправляет красивое анимационное сообщение человеку в телеграм по айди
код написан максимально криво, его можно конечно переписать, чтобы не было вечного повторения одной и той же строчки, но мне лень

1. Открыть консоль (win + r и вписываем cmd)
2. пишем по очереди: pip install pyrogram
pip install asyncio
pip install tgcrypto
3. Заходим на https://my.telegram.org/auth?to=apps
4. Входим и создаем приложение (можем писать что угодно)
5. Нам нужно только api_id и api_hash, копируем их и вставляем в код (12 строка)
6. Готово, запускаем main.py и все
